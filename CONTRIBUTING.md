# Contributing

This Community prototype accepts contributions for public modules only.

## Guidelines

- Keep the core provider interfaces stable.
- Add tests for new behavior.
- Do not add GPL/AGPL dependencies without discussion.
- Do not include private prompts, customer data, or proprietary datasets.
- Update `THIRD_PARTY_NOTICES.md` when adding dependencies.

## Suggested contribution areas

- Export plugins
- Meeting templates
- Local ASR adapters
- UI shell
- Evaluation scripts
- Documentation
